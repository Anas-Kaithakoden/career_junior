Important Terminal Commands

- List databases: \l
- Connect to a database: \c database_name
- List tables: \dt
- Exit psql: \q


CRUD Operations

- Create our own database: CREATE DATABASE blog_db;
- Create a users table:
    CREATE TABLE users (
        id SERIAL PRIMARY KEY,
        name VARCHAR(100),
        email VARCHAR(255)
    );
- INSERT:
    INSERT INTO posts(title, content, user_id)
    VALUES ('Learning SQL', 'SQL is awesome', 1);
- UPDATE:
    UPDATE users
    SET email = 'anaskk@gmail.com'
    WHERE id = 1;
- DELETE:
    DELETE FROM users
    WHERE id = 3;
- DROP:
    DROP users;
- ALTER:
    ALTER TABLE old_table_name
    RENAME TO new_table_name;


Filtering Data  

- Filter with WHERE: 
    SELECT * FROM users
    WHERE name = 'Anas';
- AND, OR
- IN
    SELECT *
    FROM users
    WHERE id IN (1,2,3);
- BETWEEN
    SELECT *
    FROM users
    WHERE age BETWEEN 18 AND 25;
-  LIKE,ILIKE(case insensitive)
    SELECT *
    FROM users
    WHERE name LIKE 'A%';
-  IS NULL, IS NOT NULL
    SELECT *
    FROM users
    WHERE email IS NULL;


Sorting & Limiting     
- ORDER BY
    SELECT *
    FROM users
    ORDER BY age ASC;
- LIMIT
    SELECT *
    FROM users
    LIMIT 5;
- OFFSET 
    SELECT *
    FROM users
    ORDER BY id
    LIMIT 5
    OFFSET 5; (6-10)


 Joins   
- INNER JOIN, LEFT JOIN
    SELECT users.name,
        posts.title
    FROM users
    INNER JOIN posts
    ON users.id = posts.user_id;

    SELECT
    p.title,
    u.name AS author
    FROM posts p
    INNER JOIN users u
    ON p.user_id = u.id;



Aggregation
- COUNT,SUM,AVG,MIN,MAX
    SELECT SUM(salary)
    FROM users;
- GROUP BY
    SELECT age,
    COUNT(*)
    FROM users
    GROUP BY age;

    SELECT u.name,
    COUNT(p.id) AS total_posts
    FROM users u
    LEFT JOIN posts p
    ON u.id = p.user_id
    GROUP BY u.id, u.name;

- HAVING
    SELECT user_id,
        COUNT(*)
    FROM posts
    GROUP BY user_id
    HAVING COUNT(*) > 1;


Constraints, Indexes & Transactions   
- NOT NULL,UNIQUE,DEFAULT,CHECK
    CREATE TABLE users (
        age INTEGER CHECK (age >= 18)
    );
- Indexes
    CREATE INDEX idx_users_email
    ON users(email);
- Transactions


Example
    SELECT u.name,
        COUNT(p.id) AS Total
    FROM users u
    LEFT JOIN posts p
    ON u.id = p.user_id
    GROUP BY u.name, u.id
    HAVING COUNT(p.id) > 0
    ORDER BY Total DESC
    LIMIT 5;
