Important Terminal Commands

- List databases: \l
- Connect to a database: \c database_name
- List tables: \dt
- Exit psql: \q


CRUD Operations

- Create our own database: CREATE DATABASE blog_db;
- Create a users table:
    CREATE TABLE users (
        id SERIAL PRIMARY KEY,
        name VARCHAR(100),
        email VARCHAR(255)
    );
- INSERT:
    INSERT INTO posts(title, content, user_id)
    VALUES ('Learning SQL', 'SQL is awesome', 1);
- UPDATE:
    UPDATE users
    SET email = 'anaskk@gmail.com'
    WHERE id = 1;
- DELETE:
    DELETE FROM users
    WHERE id = 3;
- DROP:
    DROP users;
- ALTER:
    ALTER TABLE old_table_name
    RENAME TO new_table_name;


Filtering Data  

- Filter with WHERE: 
    SELECT * FROM users
    WHERE name = 'Anas';
- AND, OR
- IN
    SELECT *
    FROM users
    WHERE id IN (1,2,3);
- BETWEEN
    SELECT *
    FROM users
    WHERE age BETWEEN 18 AND 25;
-  LIKE,ILIKE(case insensitive)
    SELECT *
    FROM users
    WHERE name LIKE 'A%';
-  IS NULL, IS NOT NULL
    SELECT *
    FROM users
    WHERE email IS NULL;
